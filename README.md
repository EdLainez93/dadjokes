# Dad Jokes - Code in Place 2026 Capstone Project 
This is my Python capstone project for the Stanford's Code in Place course for 2026.

## Reason I Made This

As a father to a three year old, I've embraced the use of dad jokes. I love hearing people groan about them, saying they're dumb, but also laughing at the punchline. It's just an overall great feeling!

It's not a very hard piece of code to write. I just utilized the API from https://groandeck.com to call for either a random dad joke or to call for the daily dad joke.
